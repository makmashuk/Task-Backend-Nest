## Installation

```bash
$ npm install

```

## DB Create

```bash
$ npm run start:dev:db
```

## ENV CONFIG

```bash
Rename the .env.example to .env
```

## Database CONFIG

```bash
Check the "src/ormconfig.ts" file and modify if any local change
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
